# MCP Server Builder

[![Python 3.8+](https://img.shields.io/badge/python-3.8+-blue.svg)](https://www.python.org/downloads/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Build Status](https://img.shields.io/badge/build-passing-brightgreen.svg)](https://github.com/alon-a/mcp-ai-agent-app-kiro-build)

An automated tool that creates complete MCP (Model Context Protocol) server environments from scratch. The MCP Server Builder handles project orchestration, template management, dependency installation, build execution, and comprehensive validation to produce production-ready MCP servers.

## üöÄ Features

### Core Capabilities
- **üèóÔ∏è Project Orchestration**: Complete project lifecycle management from creation to deployment
- **ÔøΩ  Template Engine**: Flexible template system with parameter substitution and validation
- **üì¶ Dependency Management**: Automatic dependency resolution and installation across multiple package managers
- **üî® Build System Integration**: Support for various build tools (pip, npm, cargo, go, maven)
- **‚úÖ Comprehensive Validation**: MCP protocol compliance testing and functionality validation
- **üìä Progress Tracking**: Real-time progress monitoring with detailed callbacks
- **üõ†Ô∏è Error Handling**: Robust error handling with automatic recovery mechanisms

- **üêç Python**: FastMCP and low-level SDK implementations
- **üìú TypeScript**: Official TypeScript SDK integration
- **ü¶Ä Rust**: Production-ready Rust implementations
- **ÔøΩ Go**: Foxay Contexts library integration

## üéØ Quick Start

### Create Your First MCP Server

```python
from mcp_server_builder.managers.project_manager import ProjectManagerImpl

# Initialize the project manager
manager = ProjectManagerImpl()

# Create a Python FastMCP server
result = manager.create_project(
    name="my-first-server",
    template="python-fastmcp",
    config={
        'custom_settings': {
            'server_name': 'My First MCP Server',
            'description': 'A simple example server',
            'transport': 'stdio'
        }
    }
)

if result.success:
    print(f"üéâ Server created at: {result.project_path}")
else:
    print("‚ùå Creation failed:", result.errors)
```

### Available Templates

```python
from mcp_server_builder.managers.template_engine import TemplateEngineImpl

engine = TemplateEngineImpl()
templates = engine.list_templates()

for template in templates:
    print(f"üì¶ {template.id}: {template.name} ({template.language.value})")
```

## üîß Advanced Features

### Progress Monitoring
```python
def progress_callback(project_id: str, percentage: float, phase: str):
    print(f"[{project_id[:8]}] {phase}: {percentage:.1f}%")

def error_callback(project_id: str, error: str):
    print(f"[{project_id[:8]}] ERROR: {error}")

manager = ProjectManagerImpl(
    progress_callback=progress_callback,
    error_callback=error_callback
)
```
